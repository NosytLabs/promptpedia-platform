A: Unlimited technically. But no email verification = security risk at scale.

**Q: Can we generate revenue today?**
A: No. Payment processing not implemented. Earliest: 2 weeks.

**Q: What should we build first?**
A: Phase 1 (Stripe integration) unlocks all revenue.
